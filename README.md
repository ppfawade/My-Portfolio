Welcome to my personal portfolio. This website showcases my journey bridging the gap between physical engineering and intelligent software. It features my projects, skills, and thoughts on leveraging AI to solve real-world engineering problems.

## 🚀 Tech Stack

This portfolio is built with a modern, performance-focused stack:

*   **Framework:** [React](https://react.dev/) (v18+)
*   **Build Tool:** [Vite](https://vitejs.dev/) for lightning-fast development and bundling.
*   **Language:** [TypeScript](https://www.typescriptlang.org/) for type safety and robust code.
*   **Styling:** [Tailwind CSS](https://tailwindcss.com/) for utility-first, responsive design.
*   **Animations:** [Motion](https://motion.dev/) (formerly Framer Motion) for smooth, declarative animations.
*   **Routing:** [React Router](https://reactrouter.com/) for client-side navigation.
*   **Icons:** [Lucide React](https://lucide.dev/) for clean, consistent SVG icons.
*   **Deployment:** [Vercel](https://vercel.com/) for global edge hosting.

## 🛠️ Workflow & AI-Assisted Development

This project serves as a practical example of **AI-Assisted Engineering**. The development workflow leverages advanced AI tools to accelerate coding, design, and deployment:

1.  **Ideation & Architecture:** Concepts and data structures were defined based on engineering portfolio best practices.
2.  **AI Studio & Gemini:** The core codebase was generated and refined using **Google AI Studio** powered by the **Gemini 2.0 Flash** model. This allowed for rapid prototyping of complex UI components and logic.
3.  **Iterative Refinement:** Code was iteratively improved through natural language prompting, focusing on responsiveness, accessibility, and design aesthetics.
4.  **Deployment:** The final artifact is deployed directly to **Vercel**, ensuring high availability and performance.
